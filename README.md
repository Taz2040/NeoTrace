Project Description
Our project is a telecare mobile application for pregnant mothers and newborn care, inspired by MomConnect. The goal is to help reduce maternal and infant mortality in underdeveloped communities by providing accessible digital healthcare support.

Through the app, mothers can track symptoms, receive weekly pregnancy tips, and communicate directly with healthcare professionals. The system also monitors health logs and can alert doctors if dangerous symptoms are detected.
